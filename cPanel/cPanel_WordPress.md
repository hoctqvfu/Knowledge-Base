# Cài đặt 1 trang WordPress cơ bản trên cPanel

- Trên thanh **Navigation**, chọn ```Plugins``` -> ```WordPress Toolkit```

![](./images/cp_29.png)

- Chọn ```Install``` hoặc ```Install WordPress```

![](./images/cp_30.png)

- Nhập thông tin của trang wordpress rồi chọn ```Install```

![](./images/cp_31.png)

- OK

![](./images/cp_32.png)
